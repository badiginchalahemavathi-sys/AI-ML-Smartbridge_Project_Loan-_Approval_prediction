# Loan Approval Prediction Using Machine Learning

## Project Overview
Loan Approval Prediction Using Machine Learning is a web application that predicts whether a loan application will be approved or rejected based on applicant details. The system helps banks and financial institutions make faster and more accurate loan approval decisions.

## Features
- User-friendly web interface
- Loan approval prediction
- Machine Learning-based decision making
- Real-time prediction
- Fast and accurate results

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- HTML
- CSS

## Machine Learning Algorithm
- XGBoost (or the algorithm you used, such as Random Forest or Decision Tree)

## Dataset
The project uses a loan prediction dataset containing:
- Gender
- Married
- Dependents
- Education
- Self Employed
- Applicant Income
- Coapplicant Income
- Loan Amount
- Loan Amount Term
- Credit History
- Property Area
- Loan Status

## How to Run
1. Install Python and required libraries.
2. Install dependencies:
   
   pip install -r requirements.txt
   
3. Run the application:
   
   python app.py
   
4. Open your browser and visit:
   
   http://127.0.0.1:5000
   

## Results
The model predicts whether a loan application is *Approved* or *Rejected* based on the applicant's details.
